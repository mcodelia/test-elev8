# Elev8

A responsive one-page website for Elev8, a premium creative digital studio.

## Requirements

- Node.js
- npm

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

If your local file watcher reports `EMFILE: too many open files`, use:

```bash
npm run dev:poll
```

Open the local preview:

```text
http://localhost:3000
```

Build for production:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```
